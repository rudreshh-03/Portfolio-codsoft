Hand Cricket

Hand Cricket is popularly known as the numbers in fingers. It is a 1-player as well as two player game that is developed in Android studio by using XML (eXtensible Markup Language) and Java programming language.

The main agenda of this game is:

  • User-friendly interface vs Computer and two-player
  • Easy functions and display appropriate link text
  • Coherant navigation
  • It provides clear view because there is a separate frame for each function.
  • Quick results by using Toast and user Interface.

The functions used in this game are:

  1. It is the first user interface frame when program starts. It navigates the user to choose the mode of play; ( human vs computer ) or ( human vs human )
  2. It is the toss user-interface where the decisions should be made by the players and opt to choose mode of play wheather batting or bowling.
  3. It is the batting track of the user where the used to score or chase runs by clicking the buttons numbered from 0 to 6.
  4. It is the bowling pitch where the user used to bowl and try to take wickets within the target runs by clicking the buttons numbered from 0 to 6.
  5. This is a two player user interface where ( player-1 bat and player-2 bowl ) or (player-1 bowl and player-2 bat ) consecutively.
  6. This user-interface displays the match results about the players who won.

Rules to play:

The user-interface displays number from 0 to 6. If the player and the opponent plays the same number(runs) then the current batting player is out and then the second innings begins, otherwise the batsman can play until he is out or he chased the target. then, the final result is outputted. If the batsman plays 0 and bowler plays other than 0 then the batsman's runs will be bowlers play.
